# UK House Price Analysis

## 📌 Project Overview
This project analyses UK housing market trends using publicly available datasets.  
The objective is to identify pricing patterns across regions, property types, and time periods.

---

## 🎯 Objectives
- Perform data cleaning and preprocessing
- Conduct exploratory data analysis (EDA)
- Identify regional and temporal price trends
- Create visualisations to support business insights

---

## 🛠 Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Power BI (for dashboard)

---

## 📊 Key Questions Explored
- Which UK regions have the highest average house prices?
- How have prices changed over time?
- Which property type is most expensive?
- Are there seasonal trends in housing prices?

---

## 🚀 Status
Project in progress – data cleaning and EDA phase.
